🤖 Yapay Zeka Pipeline'ı

📥 Eğitim Süreci

  • Veri: Phishing/Spam toplu corpus (binlerce örnek)
  • Önişleme: Lowercase, tokenize, stopword temizleme
  • Vektörizasyon: TF-IDF (Term Frequency-Inverse Document Frequency)
  • Model: scikit-learn classifier
  • Persist: Pickle (`.pkl`) — `mail_model.pkl`, `tfidf_vectorizer.pkl`

🔮 Tahmin Akışı

  1. WPF → HTTP POST → /predict
  2. Flask predict.py mail içeriğini alır
  3. TF-IDF vektörü üretir
  4. Model phishing/spam skoru döner
  5. Heuristik kurallar (SQL/XSS/Trojan) eklenir
  6. 0-100 arası skor → C# tarafına JSON
🛡️ Güvenlik özelliği: Model yüklenemese bile (örn. pickle uyumsuzluğu) Flask çökmez — heuristik fallback moda geçer. Sistem her zaman ayakta.

🦠 VirusTotal Entegrasyonu

API v3 Akışı

1. URL/Dosya → SHA-256 hash
2. POST /urls → upload
3. analysis_id alınır
4. GET /analyses/{id} → polling
5. stats + results parse edilir
6. VtCache'e yazılır (24 saat TTL)

Performans Optimizasyonları

  • VtCache: 24 saat aynı hash tekrar sorulmaz
  • GET-first: Önce bilinen sonucu sorgular, sonra submit
  • Rate limiter: 4 req/dakika (free tier limiti)
  • Defensive parsing: API değişimine karşı try/catch
  • Vendor breakdown: Hangi AV ne dedi, kullanıcıya gösterilir
70+ Aktif Antivirüs Motoru · ~%90 Cache Hit Oranı · <2s Cache Hit Yanıtı

📧 Gmail Entegrasyon Detayları

IMAP (Gelen)

  • imap.gmail.com:993 (SSL)
  • MailKit ile thread-safe bağlantı
  • SemaphoreSlim ile re-entrancy guard
  • HashSet ile O(1) duplicate kontrol

SMTP (Giden)

  • smtp.gmail.com:587 (STARTTLS)
  • BodyBuilder ile MIME multipart
  • Ek dosya desteği

Auto-Sync Mekanizması

DispatcherTimer ile 1/5/10/30 dakika aralıklarla:

  1. Yeni mailler IMAP'ten çekilir
  2. Duplicate kontrol (HashSet)
  3. ML modeli + VT taraması (paralel)
  4. Tehlikeli olanlar trash'e taşınır
  5. UI güncellenir (Dispatcher)

SemaphoreSlim(1,1) ile aynı anda 2 sync başlaması engellenir → race condition yok.

06

Güvenlik & KVKK

🔐 Güvenlik Katmanları

🔑 DPAPI Şifreleme

Windows Data Protection API ile Gmail App Password'ü şifrelenir. Şifre yalnızca aynı Windows kullanıcısı tarafından çözülebilir. Disk çalınsa bile parola güvende.

🔒 Single Instance Mutex

Aynı anda birden fazla uygulama kopyası açılamaz. Abandoned mutex handling ile crash sonrası bile temiz çalışır. Local\ prefix ile sadece kullanıcı bazında.

🛡️ PII Maskeleme

Veritabanına yazılan tüm loglar otomatik maskelenir: TC Kimlik, IBAN, kredi kartı, telefon, e-posta. KVKK Madde 6 uyumlu.

📜 KVKK Aydınlatma

İlk açılışta zorunlu aydınlatma ekranı. Kullanıcı kabul etmeden uygulama açılmaz. Onay ayarlar tablosunda kalıcı.

🇹🇷 KVKK Uyumlu PII Maskeleme

DatabaseHelper.MaskSensitiveData() ile veritabanına yazılmadan önce maskelenir:

⚡ Performans Optimizasyonları

📊 SQLite İndeksler

  • idx_mailler_gmail_msg_id — duplicate kontrolü O(log n)
  • idx_mailler_klasor_tarih — composite, klasör listesi
  • idx_analiz_loglari_tarih
  • idx_*_log_id — JOIN'ler için

→ 10-100x sorgu hızlanması

🔄 VT Cache (24h)

Aynı URL/hash 24 saat içinde tekrar sorulmaz. ~%90 API call azalması, rate limit korunur.

⚙️ Async/Await + Paralelizm

  • Tüm DB ops Task.Run ile
  • Network IO await edilir (UI bloklanmaz)
  • Birden fazla URL Task.WhenAll ile paralel taranır
  • SemaphoreSlim ile re-entrancy guard

🎯 HashSet O(1) Lookup

Auto-sync sırasında her mail için "bu zaten DB'de var mı?" kontrolü HashSet ile yapılır. 10K mail için 1ms.

🛠️ Karşılaştığımız Zorluklar & Çözümler

❌ Race Condition (Aynı anda 2 sync)

İlk versiyonda timer + manuel tetikleme çakışıyordu → sadece 1 mail çekiliyordu.

✓ Çözüm: SemaphoreSlim(1,1) + WaitAsync(TimeSpan.Zero) ile re-entrancy guard.

⚠️ VirusTotal API v3 Parse Bug

last_analysis_stats dokümante edilmiş ama gerçek yanıtta stats kullanılıyor.

✓ Çözüm: Defensive parsing + her iki anahtar denenir.

🐍 Python Bağımlılığı Problemi

Hedef PC'de Python kurulu olmayabilir → ML çalışmaz.

✓ Çözüm: PyInstaller ile predict.exe (75 MB bundle) → sıfır bağımlılık.

🔒 Plaintext Şifre Riski

Gmail App Password DB'de açık duruyordu.

✓ Çözüm: DPAPI ile şifreleme + otomatik migration eski kayıtlar için.
